Spotify

This project is a feature-rich Spotify clone built using HTML, CSS, and JavaScript, designed to deliver an interactive and seamless music streaming experience. With a sleek and modern UI inspired by Spotify, it ensures smooth functionality and responsiveness across desktops, tablets, and smartphones. The clone includes a fully functional music player with essential controls like play, pause, next, previous, volume adjustment, and progress tracking through a seek bar. It also features playlist management, a search bar for browsing songs and artists, shuffle and repeat playback options, and a dynamic now-playing section displaying track details such as title, artist, and album art. Hosted on a live platform, this project allows users to explore its features without requiring a local setup. The structured and well-documented codebase makes it an excellent resource for developers looking to understand responsive design and JavaScript-based interactivity in web development.

Random-Data-Generator

This project generates random user data, including names, cities, languages, salaries, and manager positions, with a simple click of the "Generate" button. Simple and interactive UI with a "Generate" button. Tech Stack: Backend: Node.js, Express.js Frontend: EJS, HTML, CSS Database: MongoDB This project showcases full-stack development skills by integrating backend logic, database operations, and dynamic frontend rendering.
